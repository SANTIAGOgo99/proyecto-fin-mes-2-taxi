Instrucciones rápidas para Render:
1. Sube esta carpeta a un repositorio de GitHub.
2. En Render crea un nuevo Web Service conectado al repositorio.
3. Build Command: pip install -r requirements.txt
4. Start Command: gunicorn app:app
5. Cuando Render genere la URL pública, pégala en URL_Render.txt, en la portada del documento y en la libreta.
